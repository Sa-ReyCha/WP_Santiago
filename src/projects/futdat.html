FutDat

Personal Project · 2025

FutDat is a real-time football analytics platform I built to predict match outcomes and generate deep tactical insights. It combines live data analysis with a personalized AI assistant that acts as a match analysis instructor — breaking down complex results into clear, actionable explanations.

Features

  • Match outcome prediction using ML models trained on historical data
  • Tactical insights derived from live match events
  • Personalized AI assistant that explains match analysis on-demand
  • Real-time data pipeline

Tech stack

  • Python for the ML models & data pipeline
  • LLM integration for the AI assistant
  • Live data ingestion

Why I built it

I love football, and I love data. FutDat was born from wanting to combine both — going beyond raw stats and building something that explains the game the way a coach would.
